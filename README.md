sproutcore-ext, an extension to sproutcore
==========================================

Install
-------

`$ bpm add sproutcore-ext` 

API
----

* SC.Object#keys()
* SC.Object#toObject()
* window.pd

see source code.

Resources
---------

* [bpm](https://github.com/bpm/bpm) a system for managing resource dependencies for client-side browser applications
